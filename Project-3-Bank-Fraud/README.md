# Bank Fraud Detection 🏦

## Project Overview
A machine learning system that detects fraudulent credit card transactions using SMOTE, Logistic Regression, and Random Forest.

## Technologies Used
- Python
- Scikit-learn
- Imbalanced-learn (SMOTE)
- Pandas & NumPy
- Matplotlib & Seaborn
- Google Colab

## Features
- Fraud detection on 284,807 transactions
- Class imbalance handling with SMOTE
- Logistic Regression & Random Forest models
- ROC-AUC comparison
- Feature importance analysis
- Confusion matrix visualization

## Model Performance

| Model | ROC-AUC |
|-------|---------|
| Logistic Regression | ~0.95 |
| Random Forest | ~0.98 |

## Visualizations
- Class Distribution (Fraud vs Normal)
- Transaction Amount Analysis
- Time-based Fraud Patterns
- Feature Correlation with Fraud
- ROC Curves Comparison
- Feature Importance Chart
- Confusion Matrices

## Dataset
- [Credit Card Fraud Detection - Kaggle](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)
- 284,807 transactions
- 492 frauds (0.172%)
- European cardholders, September 2013
