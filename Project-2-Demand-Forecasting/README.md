# Supply Chain Demand Forecasting 📦

## Project Overview
An AI-powered Supply Chain Demand Forecasting system that predicts product demand using Machine Learning (XGBoost).

## Technologies Used
- Python
- XGBoost
- Pandas & NumPy
- Matplotlib & Seaborn
- Google Colab

## Features
- Demand forecasting for 5 products
- Feature engineering with lag features
- Seasonality and trend analysis
- Interactive visualizations

## Model Performance
- Algorithm: XGBoost Regressor
- Metrics: MAE, RMSE, MAPE

## Visualizations
- Actual vs Predicted Demand
- Feature Importance Chart
- Demand Trends by Product
- Monthly Demand Heatmap

## Dataset
- Auto-generated realistic supply chain data
- Date Range: 2022-2024
- 5 Products across 5 Categories
