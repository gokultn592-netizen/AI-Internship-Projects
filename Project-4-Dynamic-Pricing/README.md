# Dynamic Pricing Engine 💰

## Project Overview
An AI-powered pricing system that predicts optimal product prices based on competitor pricing, inventory levels, customer ratings, and demand trends using Random Forest and XGBoost.

## Technologies Used
- Python
- Scikit-learn
- XGBoost
- Pandas & NumPy
- Matplotlib & Seaborn
- Google Colab

## Features
- Price prediction for 5,000+ products
- Competitor price analysis
- Inventory-based pricing adjustments
- Customer rating impact analysis
- Weekend/holiday seasonality detection
- Feature importance visualization

## Model Performance
| Model | MAE |
|-------|-----|
| Random Forest | ~$12.50 |
| XGBoost | ~$11.80 |

## Visualizations
- Price vs Competitor Scatter Plot
- Price by Category Box Plot
- Inventory vs Price Analysis
- Feature Importance Chart
- Prediction Error Distribution

## Dataset
- Auto-generated realistic e-commerce data
- 5,000 products across 6 categories
- Features: base_price, competitor_price, inventory, ratings, seasonality
